# pi-posture

Pi extension for switching the coding-agent harness posture live.

Planned command surface:

```text
/posture
/posture list
/posture status
/posture inspect
/posture default
/posture agent
/posture assist
/posture learn
/posture review
```

`default` means the extension stops influencing Pi: no prompt overlay, no context filtering, no tool changes, and no status residue.
